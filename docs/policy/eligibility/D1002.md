# ELIG_HARDSHIP_FLAG_PRESENT (D1002)

## Description
The application is declined if the applicant is flagged as currently receiving financial hardship assistance or equivalent support.

---

## Evaluation Logic
hardship_flag = true | false

---

## Threshold
hardship_flag must be false

---

## Outcome
- PASS: hardship_flag == false  
- FAIL: hardship_flag == true  

---

## Typical Remediation
- Resolve hardship status
- Reapply once hardship assistance has ceased

---

## Notes
This is a hard eligibility rule and cannot be overridden by downstream decisioning or manual review.
