# ELIG_VALID_RESIDENCY (D103)

## Description
The application is declined if the applicant does not have a valid residential address within the supported jurisdiction.

---

## Evaluation Logic
is_valid_postcode = true | false  
is_supported_region = true | false

---

## Threshold
is_valid_postcode == true  
AND  
is_supported_region == true

---

## Outcome
- PASS: both conditions satisfied  
- FAIL: either condition not satisfied  

---

## Typical Remediation
- Update residential address details
- Apply using a valid domestic address

---

## Notes
This rule is evaluated prior to any bureau or servicing checks and does not rely on external credit data.
